Wetlab


Introduction

The Washington iGEM wetlab team is in charge of molecular biology side of the project. We handled the development of a dCas9-sgRNA system designed to be inducible in yeast. Taking advantage of the violacein pathway as a proxy, we created different colors to track metabolic activity. Biobrick creation + characterization as well as Interlab were also managed by the wet lab team.

InterLab

Our software is both easy-to-use and easy to modify. User-friendliness was a high priority - much more so than similar ad hoc scientific tools. By using Oracle standard Java software development idioms and thorough JavaDoc code documentation, we’ve created a codebase that is easily adjustable by anyone with a modicum of Java experience.

Conclusion

This season, UW iGEM’s new wet lab team had quite a productive summer and fall. Having received an influx of new members we happily trained and worked hard alongside one another. For our main project were able to create the sgRNA expression system and test it as working in yeast. We managed to create and characterize two novel inducible promoter biobricks for yeast. And this year, for the first time ever, we were able to participate in the iGEM 2017 Interlab collaboration alongside other iGEM teams across the globe.
